# Link to EM Metamaterials

Acoustic metamaterials and electromagnetic metamaterials use the same engineering idea:

> design the unit cell to control wave propagation.

| Acoustic metamaterial | EM metamaterial |
|---|---|
| sound pressure \(p\) | electric/magnetic fields \(E,H\) |
| acoustic duct / tube | waveguide or free space |
| side branch / cavity | LC resonator, split-ring resonator, patch resonator |
| Bragg scattering from spacing | Bragg scattering in photonic crystals / periodic dielectrics |
| sound stop band | EM bandgap / FSS stop band |
| resonant absorber | radar absorber / Salisbury screen / FSS absorber |

The material response is determined by

\[
\text{behavior} = \text{constituent material} + \text{unit-cell geometry} + \text{periodicity}
\]
