# Bragg Scattering

Bragg scattering occurs when waves encounter a repeated structure. Each cell reflects a small amount of the wave. If the spacing is right, those reflections add together and reduce transmission.

A first estimate for the Bragg stop-band center is

\[
a \approx \frac{\lambda}{2}
\]

Using \(\lambda = c/f\),

\[
f_\text{Bragg} \approx \frac{c}{2a}
\]

## Geometry meaning

In this app, Bragg scattering is shown as **pipe-width modulation**:

- wide section width \(W\)
- narrow section width \(w\)
- cell period \(a\)

### Design intuition

- Larger spacing \(a\) blocks lower frequencies.
- Smaller spacing \(a\) blocks higher frequencies.
- Larger difference between \(W\) and \(w\) gives stronger reflection.
- More repeated cells usually produces a deeper stop band.

## Example

If \(a = 0.17\,\text{m}\),

\[
f_\text{Bragg} \approx \frac{343}{2(0.17)} \approx 1009\,\text{Hz}
\]
